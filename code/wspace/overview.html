Core Features of the Protocol

Understanding the key components that make our governance system secure and effective

Deposit & Lock Funds

Securely deposit ADA into a smart contract vault that requires multiple approvals for any withdrawal. Your funds are protected by blockchain-enforced rules that cannot be bypassed.

  • Set custom approval thresholds (1-2 officials)
  • Define authorized officials' addresses
  • Configure time-locked deadlines
  • Automatic refund protection

Multi-Signature Approval

No single person can move funds unilaterally. Multiple authorized officials must approve each transaction, distributing power and eliminating single points of failure.

  • Require 1 or 2 approvals per transaction
  • Each approval recorded on-chain
  • Real-time approval tracking
  • Immutable approval history

Fund Release

Once the required approvals are collected before the deadline, funds can be released to the designated recipient address. The smart contract automatically verifies all conditions.

  • Automatic threshold verification
  • Deadline enforcement
  • Instant fund transfer
  • Complete transaction transparency

Automatic Refunds

If the approval threshold isn't met before the deadline, the original owner can safely claim a full refund. This protects against indefinite fund lockup.

  • Time-locked refund mechanism
  • Owner verification required
  • Full fund recovery
  • No approval needed for refunds
